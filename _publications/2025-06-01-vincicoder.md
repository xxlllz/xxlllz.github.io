We propose VinciCoder, a unified framework for multimodal code generation that addresses the limitation of single-task training in existing VLMs. VinciCoder uses a two-stage approach: (1) large-scale supervised fine-tuning with 1.6M image-code pairs, and (2) Visual Reinforcement Learning with a novel coarse-to-fine reward mechanism using DINOv2-L for visual feedback. The model achieves state-of-the-art performance on diverse benchmarks including chart-to-code, webpage generation, SVG, LaTeX, and chemical molecule visualization.
