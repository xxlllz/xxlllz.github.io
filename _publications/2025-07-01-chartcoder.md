We propose ChartCoder, the first dedicated multimodal large language model specifically designed for chart-to-code generation. ChartCoder leverages Code LLMs as the language backbone and introduces Snippet-of-Thought (SoT) for step-by-step generation. The model achieves superior performance on chart-to-code benchmarks with only 7B parameters, surpassing existing open-source MLLMs in both chart restoration quality and code executability.
